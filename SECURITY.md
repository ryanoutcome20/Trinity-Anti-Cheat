# Security Policy

## Supported Versions

All versions of the latest build (both the development and main branch) are supported. If you find a security issue in either of these branches you may report it.

## Reporting a Vulnerability

There are a couple of types of vulnerability we want privately diclosed:

* RCE Vulnerabilities: Including console command execution, serverside code execution, etc.
* Networking Vulnerabilities: Specifically those involving serverside logging, database logging, file I/O, or sanitization.
